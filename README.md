# 99bottles_js

Kata from "99 Bottles of OOP":
Book: https://sandimetz.com/99bottles

## Getting started
1. Run 
`yarn install` or `npm install`, 
2. Then run the tests with `yarn test` or `npm test`
3. Unskip one test at a time and fix!
